# 企业微信配置指南

本文档说明如何配置企业微信机器人以接入 OpenCode Bridge。

---

## 1. 配置概览

配置企业微信机器人需要完成以下步骤：

1. 创建企业微信应用
2. 配置机器人
3. 获取 Bot ID 和 Secret
4. 配置消息接收地址
5. 在 Bridge 中填写配置参数

---

## 2. 创建企业微信应用

### 步骤 1：访问管理后台

访问 [企业微信管理后台](https://work.weixin.qq.com/)

### 步骤 2：创建应用

1. 进入"应用管理"
2. 点击"创建应用"
3. 上传应用 Logo，填写应用名称
4. 选择可见范围
5. 点击"创建"

---

## 3. 配置机器人

### 3.1 添加机器人

1. 在应用详情页面，找到"机器人"选项
2. 点击"添加机器人"
3. 设置机器人名称
4. 获取 **Webhook URL**

### 3.2 配置 API 权限

1. 在应用详情页，进入"权限管理"
2. 添加必要的 API 权限
3. 获取 **Secret**

---

## 4. 获取配置信息

### 4.1 获取 Bot ID（AgentId）

在企业微信后台，从机器人信息中获取 Bot ID（AgentId）。

### 4.2 获取 Secret

在应用详情页 → 权限管理 中获取 Secret。

### 4.3 配置消息接收地址

1. 在应用详情页找到"接收消息"配置
2. 设置 API 接收地址为：
   ```
   http://your-server:your-port/wecom/webhook
   ```

---

## 5. Bridge 配置

在 Web 面板或 `.env` 中配置以下参数：

| 参数 | 必填 | 说明 |
|------|------|------|
| `WECOM_ENABLED` | ✅ | 设置为 `true` 启用企业微信 |
| `WECOM_BOT_ID` | ✅ | 企业微信 Bot ID（AgentId） |
| `WECOM_SECRET` | ✅ | 企业微信 Secret |

---

## 6. 验证

配置完成后，在企业微信中验证：

1. 向机器人发送消息
2. 如果配置正确，机器人应该会响应
3. 测试命令如 `/help`、`/panel`

---

## 7. 故障排查

### 机器人无响应

1. 检查 `WECOM_ENABLED` 是否为 `true`
2. 检查 Bot ID 和 Secret 是否正确
3. 检查消息接收地址是否正确配置
4. 查看 Bridge 服务日志

### 消息接收地址配置错误

1. 确认 Webhook URL 配置正确
2. 检查企业微信后台回调配置
3. 确保服务器可以访问外网

### 应用权限不足

1. 检查企业微信应用权限设置
2. 确保添加了必要的 API 权限

---

## 8. 限制说明

企业微信适配器有以下限制：

- 使用纯文本消息交互
- 不支持富文本卡片
- 文件发送受限于企业微信 API 限制

建议在测试群组中先验证配置。

---

## 9. 安全建议

1. **Secret 保护**: 不要泄露应用 Secret
2. **权限最小化**: 只授予必要的权限
3. **访问控制**: 限制应用可见范围
4. **定期审计**: 定期检查应用权限和使用情况
