# 发行说明：v2.9.5-beta

**发布日期**: 2026-03-23

---

## 相比 v2.9.3-beta 的主要变更

---

### 🎯 核心特性

#### 1. 多平台支持扩展

新增支持以下平台：
- **企业微信 (WeCom)** - 完整消息处理能力
- **Telegram** - 推送与交互能力
- **QQ** - 基础消息接入
- **WhatsApp** - 基础消息接入
- **个人微信 (Weixin)** - 私人微信登录能力

#### 2. 平台能力对齐

- 全平台能力对齐飞书 baseline
- 统一平台抽象层 (`PlatformSender`)
- 支持按平台启用/禁用控制

### 🚀 升级指南

#### 升级步骤

```bash
# 1. 拉取最新代码
git pull origin beta

# 2. 安装依赖
npm install

# 3. 构建 Web 前端 (如有变更)
cd web && pnpm install && pnpm build

# 4. 重启服务
npm run start
```

#### 配置新平台

1. 访问 `http://localhost:4098` 进入配置面板
2. 在「平台配置」中配置对应平台凭据
3. 在「核心路由」中启用对应平台

---

### ⚠️ 注意事项

1. **平台凭据**：新增平台需在对应平台申请开发者凭据
2. **国内镜像**：系统会根据时区自动判断是否使用国内镜像源
3. **Windows 用户**：已修复多项 Windows 兼容性问题
